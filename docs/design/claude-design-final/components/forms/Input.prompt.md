Base labeled text field. Error replaces helper text below the field; never both at once.

```jsx
<Input label="التاجر" placeholder="مثال: بقالة العائلة" required />
<Input label="البريد الإلكتروني" icon="mail" dirOverride="ltr" />
```
Use `dirOverride="ltr"` for email/URL/technical fields so Latin text doesn't reverse.
