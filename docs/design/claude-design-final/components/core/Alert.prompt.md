Inline page/section-level banner — distinct from Toast (which floats and auto-dismisses).

```jsx
<Alert tone="warning" title="مفتاح API قارب على الانتهاء">يرجى تحديث مفتاح المزوّد قبل 3 أيام.</Alert>
```
Use `error` for AI extraction failures with a specific next step in the body text — never a generic "something went wrong".
