Icon-only button for toolbars, table row actions, and header controls. `label` is mandatory — it becomes `aria-label`.

```jsx
<IconButton icon="pencil" label="تعديل" />
<IconButton icon="trash-2" label="حذف" variant="filled" />
```
`lg` (44px) is the default for any mobile-reachable icon button per the 44×44px touch-target rule.
